# 真实资料题库 v1

更新时间：2026-07-19

## 1. 定位

真实资料题库是 Study Planner 的题目来源层。

它的目标不是替代学习计划，而是让每日练习、阶段测试、错题复盘优先使用用户提供的真题 / 教材 / 学习包内容。

当前产品重心：

```text
真实资料 / 真题 / 教材原题
→ 系统练习与判题
→ AI 辅助中文解释、错因分析、复习建议
→ 错题集与明日重点
```

AI 不作为主题库来源。AI 只做：

- 中文解释
- 答题解析
- 错因归纳
- 生词与语法点整理
- 缺题时生成少量同型补充题

## 2. 题目来源优先级

题目来源按以下优先级使用：

1. 用户已购买 / 自有的真题、教材、学习包
2. 用户上传的错题照片、截图、PDF
3. 官方公开样题或公开题型说明
4. AI 按真实资料模板生成的同型补充题
5. 本地兜底题

每日主练习应优先使用真实资料题。AI 同型题只适合补量，不应伪装成真题。

### 3.2 严格任务路由

真实题组必须同时匹配考试、等级、模块和当前任务训练点，不能只因同属“听力”或“阅读”就复用。

- 第 102 届 TOPIK II 看图题只匹配“看图听关键词”或“看图与图表理解”。
- 第 102 届 TOPIK II 对话续句与行动题只匹配“听力 · 判断下一步行动”或“听内容一致”。
- 第 102 届 TOPIK II 短信息阅读题只匹配“通知公告阅读”“促销广告阅读”或“图表信息读取”。
- TOPIK I 标识与公告题只匹配“公告信息读取”“广告信息读取”或“标识与公告理解”。
- “听数字和时间”“听否定和时态”等任务不得复用第 102 届看图题的图像或原始 MP3；未匹配时回到对应训练点自己的题目。

### 3.3 question

每道题必须至少包含：

- `materialQuestionId`：稳定题目 ID
- `materialImage`：答题前可展示的原题图，只能指向不含中文讲解的题目图
- `stem`：韩文题目要求
- `stemZh`：中文题目要求
- `passageZh`：原题内容中文释义，答题后展示
- `options`：韩文选项
- `optionTranslations`：中文选项解释，答题后展示或用于解析
- `answer`：0 起始答案索引
- `answerZh`：正确答案中文意思
- `explanation`：韩文或原始解析
- `explanationZh`：中文答题解析
- `source`：页码 / 资料位置

## 4. 图片规则

答题前只能展示原题内容，不展示中文翻译、答案、解析或必背词。

允许：

```text
assets/materials/.../question/question-100.png
```

禁止作为答题前图片：

```text
assets/materials/.../block-cn/block-100.png
```

`block-cn` 这类图片只能作为内部校对材料或答题后讲解素材，不能直接出现在题目区。

## 5. OCR / 人工结构化流程

正式导入资料时按以下步骤走：

```text
原始图片 / PDF
→ OCR 提取韩文题干、选项、答案和页码
→ 模型生成中文释义与解析
→ 结构校验
→ 抽样人工检查
→ 进入真实资料题库
```

模型校验建议：

- OpenAI 作为主识别 / 主解释模型
- DeepSeek 作为辅助复核模型
- 两个模型不一致时标记为 `needs-review`
- 大批量导入不依赖人工逐题检查，但必须抽样复核

## 6. 验收规则

每批导入必须满足：

- 每道题有稳定 ID
- 图片路径存在
- 答题前图片不包含中文翻译或解析
- 选择题必须有 4 个选项
- `answer` 必须落在选项范围内
- 必须有中文题目要求、原题释义和答题解析
- 题目必须标注来源资料和页码 / 图片序号
- 不得混入和当前任务类别不一致的题
